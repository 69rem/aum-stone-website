# AUM Stone Website

Commercial landing page for a premium stone countertops company.  
This project replaced the company's existing website with a modern, responsive solution that I deployed to production.

**Live website:** [https://aum-stone.ru/](https://aum-stone.ru/)

## Overview
A complete website overhaul for AUM Stone, featuring a responsive design, interactive elements, and improved user experience. Successfully migrated from their old hosting to Beget with improved performance and modern features.

## Features
- **Responsive Design** - Mobile-first approach, fully adaptive layout
- **Consultation Form** - Client-side validation with secure backend processing
- **Project Gallery** - PhotoSwipe integration for smooth image viewing
- **Interactive Map** - Yandex Maps with location and contact details
- **Custom Components** - Vanilla JavaScript slider, modal windows, navigation
- **Accessibility** - Semantic HTML, ARIA attributes, keyboard navigation
- **Performance** - Optimized assets, lazy loading, SCSS preprocessing

## Tech Stack
**Frontend:**
- HTML5, Semantic markup
- CSS3 with Grid, Flexbox, Custom Properties
- SCSS for structured styling and variables
- Vanilla JavaScript (ES6+ modules)

**Integrations:**
- PhotoSwipe for gallery functionality
- Yandex Maps API
- Telegram notification system (tokens secured on server)

## Development Notes
- Used SCSS for maintainable, structured CSS with variables and mixins
- Implemented mobile-first responsive design principles
- All sensitive API tokens and credentials removed from this repository
- Production deployment handled via Beget hosting with proper configuration
- Form validation includes both client-side and server-side checks

## Project Status
Completed and deployed to production. All features are fully functional and the website is currently serving live traffic.

## Security
This repository contains a clean version with all sensitive data removed. Production credentials are managed securely on the hosting server.
